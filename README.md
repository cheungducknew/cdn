https://cdn.jsdelivr.net/gh/cheungducknew/cdn@latest/

```
git add . 
git commit -m "update" 
git push -f		   # force
```

https://cdn.jsdelivr.net/gh/cheungducknew/cdn@master/img/  
中的cdn改成Purge，如
https://purge.jsdelivr.net/gh/cheungducknew/cdn@master/img/
